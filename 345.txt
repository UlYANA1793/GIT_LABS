Проверка состояния рабочей папки осуществляется командой
$ git status .
В ответ Git Bash покажет название текущей ветви «master», присвоенное
по умолчанию, и список всех неотслеживаемых файлов в рабочей папке, не
подпадающих под шаблоны файла «.gitignore». Имена этих файлов будут
выделены красным шрифтом.
